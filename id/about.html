Tentang Saya

Saya Acep Triana, S.Kom

Saya orang Bandung, Jawa Barat. Saya memiliki ketertarikan mendalam pada arsitektur sistem, keamanan siber, dan teknologi finansial. Perjalanan karier saya mencakup spektrum teknologi yang luas. Di bidang pengembangan web, saya terbiasa membangun antarmuka dan sistem backend yang tangguh menggunakan HTML, Tailwind CSS, PHP, Node.js, dan React.

Di balik layar, saya berperan sebagai System Administrator dan Network Engineer yang mengelola server (Linux/Windows), orkestrasi Docker, serta konfigurasi jaringan (Cisco & MikroTik). Selain infrastruktur konvensional, saya aktif terjun ke dunia Web3, DeFi, dan pasar finansial, memadukan teknologi dan finansial melalui pengembangan bot crypto serta sistem trading otomatis (Expert Advisor) Metatrader.
